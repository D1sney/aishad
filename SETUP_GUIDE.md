# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка API ключей

Скопируй `.env.example` в `.env` и заполни:

```env
TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
OPENROUTER_API_KEY=твой_ключ_openrouter
```

## 3. Настройка администратора

1. Запусти бота:
```bash
python main.py
```

2. Найди бота в Telegram и отправь `/myid`
3. Останови бота (Ctrl+C)
4. Добавь свой ID в `config.json`:
```json
{
  "admin": {
    "user_ids": [твой_id_здесь]
  }
}
```

5. Запусти бота снова:
```bash
python main.py
```

## 4. Настройка данных

Отредактируй `data/data.txt` - добавь информацию о ШАД.

## 5. Использование

### Обычные пользователи:
- Просто пиши вопросы боту

### Администраторы:
- `/config` - посмотреть настройки
- `/set_top_k 5` - изменить количество chunks (применяется сразу)
- `/set_chunk_size 500` - изменить размер chunks (требует перезапуска)
- `/add_admin 123456` - добавить нового админа

## Рекомендуемые настройки для старта

В `config.json`:
```json
{
  "rag": {
    "chunk_size": 400,
    "top_k": 5,
    "embedding_model": "all-MiniLM-L6-v2"
  }
}
```

Эти значения обеспечивают хороший баланс точности и стоимости.
